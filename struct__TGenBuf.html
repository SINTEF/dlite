Detailed Description

Buffer for generated output. Example use that prints "Hello world!" to stdout: 

TGenBuf s;
tgen_buf_init(&s);
tgen_buf_append(&s, "Hello", -1);
tgen_buf_append_fmt(&s, " %s!", "world");
printf("%s\n", tgen_buf_get(&s));
tgen_buf_deinit(&s);
