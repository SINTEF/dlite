Create a unique temporary file.

Parameters
dir(optional) directory to create file. If NULL use default TMP directory.
prefix(optional) prefix for file name. If NULL use "tmp.".
pathname(optional) pointer to a buffer to receive the temp filename. Allocated using malloc(); user to free. Ignored if NULL.
keepIf keep is nonzero and pathname is not NULL, then keep the file after closing. Otherwise file is automatically deleted when closed.
Returns
Pointer to stream opened in binary read/write (w+b) mode, or a null pointer on error.
Exceptions
ENOMEMNot enough memory to allocate filename.
